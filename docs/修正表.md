## 状態の凡例

- `未着手` … 登録のみ
- `プラン済` … 修正プラン作成・承認待ち／承認済み
- `修正中` … Codex にて実行中
- `未確認` … 実装済みだが実機での確認が済んでいない
- `保留` … 対応策検討済みだが実装は見送り中
- `完了` … 修正・動作確認済み
