# もういっそ敗けてくれ — プレイヤー向け説明書（v0.6.1）

プレイファイル: `mou_isso_v0_6_1.html`  
ブラウザで開き、更新後は **Ctrl+F5**。

---

## あなたの役割

4人の皇子の補給係です。手紙を読み、物資を作り・買い・送り、前線を進めます。

---

## 1ターンの流れ

1. **手紙** … 読む・返事・（任意）謁見  
2. **経営** … 市場・工房・牧場・薬房  
3. **輸送** … 手渡し or 前線輸送  
4. **結果** … 完成・出来事・次の季節へ  

1ターン ＝ 春→夏→秋→冬 の1季節。

---

## 手紙フェーズ

### 皇子タブ

- 上の行: **出立待ち** / **戦地壱〜参** / **敵城**  
- 下の行: 新しい手紙があるときだけ赤の **`[新着！]`**  

### 新着があるとき

- 文官の報告（依頼タブ）と娘の手紙  
- 返事を選ぶ（取り消し不可）  

### 新着がないとき

- 娘が「前回のやりとりはこんなでした」と見せます  
- 無理に返事しなくても大丈夫です  

### フェーズ完了

- 未返信の新着があるとき、確認が出ます  
  - **OK** → 経営へ  
  - **キャンセル** → 手紙に戻る  

### 謁見（進言）

- **300両**かかります（ボタンに表示）  
- 1ターンに1つ。効果は **今季から**  
- 例: 店の物資増産、工房・薬房の増員、市場強化、厩舎、名馬探索、皇子を褒める  

---

## 経営フェーズ

### 市場

- 店の在庫と価格は季節で動きます  
- **同じ季節に買ってすぐ売る**のは損しやすいです  
- **安い季節に仕入れて、足りない季節に売る**と利益が出やすいです  
- 工房・薬房で作った完成品（剣・弩・投石機・名剣・投刀・暗刀・雲梯・爆弾・火槍・衝車・回復薬・火薬・しびれ薬・睡眠薬・仙丹）は **売るだけ**（店では買えません）  

### 工房・薬房

- 枠の数だけ生産指示  
- 完成は結果フェーズ。輸送は **翌ターン以降**  
- 余った完成品は市場で売れます  

### 牧場

1. 初回は迷い馬イベント（雄1・雌1）  
2. 春に仔馬が生まれるとポップアップで報告  
3. 秋につがい、冬に餌（兵糧・回復薬）  
4. 出荷解禁後は一覧から馬を売却可（★が高いほど高値）  

---

## 輸送フェーズ

### 出立前

手渡しで準備品を渡します。足りないと出立できません。  
1ターンに出立できる皇子は **1人**です。

### 出立したあと

#### 兵糧太・攻三城・武刃四

- 輸送手段: **人足 / 荷車 / 船 / 馬車**（条件付き）  
- **馬を手段には使いません**  
- 画面に  
  - 依頼の進み具合（何個納入したか）  
  - 現庫・依頼残・**輸送中**  
  - 手段を選ぶと **「ノード○まであと○T」**  

| 手段 | いつ使える | 目安 |
|------|------------|------|
| 人足 | 最初から | 遅い・小荷 |
| 荷車 | 兵糧が戦地壱を突破 | 積載2 |
| 船 | 兵糧が戦地弐を突破 | 速い・積載4 |
| 馬車 | 騎馬次が戦地弐突破 **かつ** 荷車解禁 | 速2・積載2 |

人足は最大4人。荷を送っている間は戻りません（到着までのTが終わると戻る）。

#### 騎馬次

- 出立前: 手渡しのみ  
- 出立後: **馬の一覧から送る馬を選ぶだけ**（自走。手段ボタンなし）

## お金の稼ぎ方

1. 工房・薬房で作ったものを売る  
2. 馬を育てて売る  
3. 市場で **季節をまたいだ** 仕入れ・売却  

進言は300両かかるので、計画的に。
